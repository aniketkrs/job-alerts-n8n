# Job Alerts Automation (n8n)

This repository contains an n8n workflow that:
- Checks Gmail every 4 hours
- Identifies Product Management Intern and Associate Product Manager roles
- Sends job alerts via WhatsApp (Twilio) and Email

## Tech Used
- n8n
- Gmail API
- Google Gemini
- Twilio WhatsApp

## How it works
The workflow uses a scheduled trigger, Gmail integration, AI-based filtering, and notification nodes.
